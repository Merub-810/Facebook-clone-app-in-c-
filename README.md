# Project
It's a Facebook Clone app in C++ that perform following functionalities:
<ol>
<li>Sign up
<li>Login</li>
<li>Add Post</li>
<li>View Posts</li>
<li>Delete Post</li>
<li>Comment on Post</li>
<li>Veiw Comments</li>
<li>Add Friends</li>
<li>Display Friend Request</li>
<li>Display Friend List</li>
<li>Messaging</li>
</ol>
